A variety of different geo shapes as GeoJSON

## Used tools
Mapshaper http://mapshaper.org/

QGIS http://qgis.org/en/site/

GDAL ogr2ogr http://www.gdal.org/ogr2ogr.html
`ogr2ogr -f "GeoJSON" file.geojson file.shp`
